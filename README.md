# 日文學習網站

一個從零開始的日文學習網站，涵蓋 JLPT 文法解說與課本單字，介面為繁體中文。

## 🔗 線上網站

**https://oliverhuang20050304.github.io/japanese_website/**

## ✨ 功能

- **文法解說**：N5、N4 核心文法，以及課本進度（第 17～20 課）
  - 每條含詳細解說、接續方式、例句
  - 例句漢字皆標假名（ruby 注音）
  - 點例句可用瀏覽器語音**朗讀日語**
- **單字**（第 17～20 課）
  - 可切換「中日對照／只顯示日文／只顯示中文」
  - 隱藏時**點單張卡片**即可單獨翻開並朗讀
  - 附詞性、假名注音
- **小測驗**：每個分類／每課可即時測驗
- **假名注音開關**：可隱藏注音自我測驗
- **搜尋**：可搜文法句型、單字（日文／假名／中文）
- **響應式設計**：手機、電腦皆適用

## 🛠 技術

單一 `index.html` 檔，純 HTML／CSS／JavaScript，無外部相依，離線亦可開啟。

## 📝 更新方式

修改 `index.html` 後：

```bash
git add -A && git commit -m "更新" && git push
```

推上去約一分鐘後，GitHub Pages 線上版會自動更新。
